## 📝 Descripción
Es un sistema cuyo objetivo es permitir al usuario reservar una habitación de hotel, indicando cantidad de personas, número de habitación deseada, datos personales, noches a hospedarse y servicios adicionales deseados.

3. Hacerle click a Run Python File situado en la esquina derecha superior en el Visual Studio Code.
